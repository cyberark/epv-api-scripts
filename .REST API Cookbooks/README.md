# CyberArk REST API Cookbooks

This repository contains practical, example-driven cookbooks for automating CyberArk operations using the REST API. Each cookbook provides step-by-step PowerShell scripts and command-line examples for common account, safe, and system management tasks.

## Examples Of What You'll Find
- Account management (create, update, delete, search)
- Safe management (create, update, remove)
- Safe member management
- Password management
- System health and monitoring
- Access request management
- SSH key management
- Example scripts in PowerShell, Bash, and Python

## Getting Started
1. Review the official CyberArk REST API documentation at [Privileged Access Manager - Self-Hosted](https://docs.cyberark.com/pam-self-hosted/latest/en/content/resources/_topnav/cc_home.htm) or [Privilege Cloud](https://docs.cyberark.com/privilege-cloud-shared-services/latest/en/content/resources/_topnav/cc_home.htm)
2. Choose the cookbook for your use case
3. Copy and adapt the example scripts for your environment

## Structure
- **PowerShell**: Step-by-step scripts for all major API operations
- **Bash & Python**: Example snippets for cross-platform automation

## Best Practices
- Always secure your API credentials and tokens
- Handle errors and edge cases in automation scripts
- Use HTTPS for all API calls
- Refer to CyberArk documentation for API changes and updates
